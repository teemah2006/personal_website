Description

StudyEase AI is a web application designed to help students organize their studies more effectively by generating personalized study plans and learning recommendations using artificial intelligence. The platform tailors each plan based on user-specific inputs such as subjects, topics, weaknesses, exam type, and time availability.

Key Features

User-Centered Onboarding
Implemented a structured sign-up process to collect relevant details (exam type, education level, goals), ensuring the AI generates highly personalized study plans.

AI Prompt Engineering
Designed and optimized AI prompts to deliver practical, tailored study strategies instead of generic advice, improving overall user satisfaction.

YouTube API Integration
Automatically suggests high-quality educational videos aligned with selected topics for enhanced learning.

Motivational Email Reminders
Sends automated motivational emails 30 minutes before each scheduled study session to encourage consistency and combat procrastination.

Exam Type & Education-Level Adaptation
Supports multiple exam systems (e.g., WAEC, JAMB, SAT, IGCSE) with flexible academic term/semester structures, making the app useful across different countries.

Firestore Database
Built secure and scalable data storage for managing user profiles, preferences, and study progress.

UI/UX Enhancements
Designed a clean, responsive, and intuitive interface with Next.js and Tailwind CSS, improving accessibility across devices.

Scalable Architecture
Structured the project to accommodate future features like collaboration, live study sessions, and progress tracking.
